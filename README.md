
# TD4 – CSS Units, Box Models, Fonts

## Overview

This project is created as part of the **MERN Stack course assignment (TuteDude)**.
The goal of this task is to understand and apply **CSS Units, Box Model concepts, and basic Font styling** by building a simple webpage layout.

The page contains an image and a button placed inside a container. The layout uses **viewport units** to control spacing and ensures the page does not scroll.

---

# Assignment Requirements

## 1. HTML Setup

* Define a proper HTML document structure.
* Set the document title.
* Link an external CSS file.

---

## 2. Body Content

* Create a **div container** for displaying the image.
* Insert an **image inside the div**.
* Add a **button below the image** for user interaction.

---

## 3. CSS Styling

The following styling requirements were implemented:

* Give the section **equal spacing of 25% from the left and right side of the screen**.
* Keep spacing from **top and bottom as 10% of the screen**.
* Customize the **button appearance using gradient background**.
* Apply **font settings** to improve readability.
* Use **Box Sizing property properly**.
* Ensure the **page does not go into scroll**.
* Use **Viewport units (vh, vw)** for layout spacing.

---

## Restrictions (Important)

The following CSS properties were **not used as per assignment instructions**:

* Flexbox
* `overflow: hidden`
* `position: relative`
* `position: absolute`

---

## CSS Concepts Used

* CSS Units (`vh`, `vw`, `%`)
* CSS Box Model

  * Margin
  * Padding
  * Border
  * Box-sizing
* Font Styling

  * `font-family`
  * `font-size`
* Gradient Background

  * `linear-gradient()`
